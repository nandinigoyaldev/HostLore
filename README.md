# HostLore 🖥️

> **The Hosting Bible — But Actually Fun**

A premium, interactive web hosting education platform. Learn the internals of web hosting, DNS, databases, and deployment platforms — in a way that actually sticks.

---

## ✨ What's Inside

| Section | What You'll Learn |
|---|---|
| 🌐 **The Journey of a Click** | Every step from typing a URL to the page rendering (8 stages, all explained) |
| 🏠 **Hosting Types** | Shared → VPS → Dedicated → Cloud → Serverless with real analogies |
| 🔱 **The Database Trinity** | SQL vs NoSQL vs NewSQL — why all three exist and when to use each |
| ⚔️ **Platform Wars** | Vercel, Netlify, Railway, Render, AWS, GCP, Azure + 8 more platforms compared |
| 🤔 **Ask HostLore** | Smart Q&A — jumps to relevant sections OR auto-creates a GitHub Issue |

---

## 🚀 Features

- **Animated Terminal** — Live command sequences showing DNS, database, and deployment concepts
- **Interactive Learning** — Scroll-triggered animations, counter stats, step-by-step journey cards
- **Ask HostLore** — Smart keyword matching; if a topic isn't covered yet, it opens a pre-filled GitHub Issue requesting that feature
- **Premium Dark UI** — Glassmorphism, gradient accents, smooth micro-animations, fully responsive
- **SEO Optimized** — Proper meta tags, semantic HTML5, accessible with ARIA labels
